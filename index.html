CSS3 Buttons

Default

To acheive this look you should use either:

<a class="btn">Login<a>

or

<button class="btn">Login<button> Login

Size - mini, large, heavy

To achieve small buttons you should use:

<a class="vtn mini">Mini<a>

or

<button class="vbtn large">Large<button> Mini

You can also make the button text embolden by using the "heavy" class:

<a class="vbtn heavy">Heavy<a> Heavy
